# 跑步

在大学时，我常常为未来感到焦虑不安，每周都要绕学校跑好几圈。跑步可以让我们内心平静下来，减轻压力，缓解抑郁情绪。虽然跑了那么长时间，但我发现自己从没有爱上过跑步。工作以来，我比大学快乐了许多，再加上周边适合跑步的环境比较少，当时雾霾也挺严重，于是我不再坚持这项运动了。

想要跑步的朋友，可以看看[Papers目录中的英文论文](Papers/)，也可以看看Youtube上的视频，你没有必要光脚跑步，可以考虑穿帆布鞋甚至体操鞋，不要攀比距离，尊重自己的感受。除了跑步外，还有很多运动可以快速燃烧卡路里。

![](images/impact.jpg)

跑步时尽可能不要穿带减震的跑鞋，那只会让你养成不良的习惯！脚掌前/中部先落地时，小腿的动能会因为其随着落地点转动而被消耗掉，因而对膝盖的冲击会非常小。
