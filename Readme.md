# 徒手健身

健身时，我总是追求尽可能有实用价值，尽可能少或不用其它专业器械。**囚徒健身/Convict Conditioning**对我影响很深，但我并没有完全遵守上面的健身计划，我不想让锻炼成为一件痛苦的事情，让健身变得快乐有趣，才有可能长期坚持下去。

2017年以前我的上半身肌肉很弱，连一个标准的正手引体向上都做不了，越是做不了就越不愿到单杠下丢人现眼，我感觉大多数人的心态应该跟我差不多。做了程序员后，刚开始白天上班晚上还会玩电脑，但很快就发现腰部、后背、脖子、肩膀、手腕时不时会出现不适，我曾经想做一个很厉害的黑客，慢慢我醒悟过来，没有好的身体技术水平再好也不会快乐的。

这几年来，我慢慢探索出一种让自己既快乐又变强的健身方式，做成了许多曾经不敢想象的动作。这个项目包含我的健身经验，受伤以及恢复过程，也会附带一些我参考过的资料，希望对大家有一点帮助。
