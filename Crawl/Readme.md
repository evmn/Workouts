# 爬行

现在城市的生活方式非常不利于人们的健康，人们每天运动时间相比过去急剧下降，我们亟需改变自己的健身方式。

走路和跑步虽然也是不错的运动方式，但是爬行也许性价比更高。通过爬行，我们全身大多数平时忽视的部位都可以锻炼到，每天爬上几百步可能比跑几千步还要累。

我感觉爬行可能会缓解人们腕部、颈部、肩部、腰部的不适，增强腿部及核心的力量，当然还有待长期观察。
