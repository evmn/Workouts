# 轮滑

轮滑运动有一定的危险性，建议先通过**囚徒健身**中的*六艺*把身体素质提上去，再去尝试。

轮滑最好配合着其它运动一起练，练前适当热身，练后拉伸一下，并且要不时光脚锻炼下足部肌肉。
