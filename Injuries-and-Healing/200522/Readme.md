## 不明原因胸痛

根据我的日记，我在2019～2020年总共应该出现过两次短暂的胸痛，几次心悸。当时的结论是过度疲劳、休息不足、锻炼过度。但是另外一次，我不知道是什么时候，日记里没有写。

下面是5月22日当天下午5点左右的日记：

>刚刚毫无征兆的感觉胸部一阵刺痛，感觉都不太敢扩胸和用力呼吸，不知道是什么情况。回宿舍休息片刻，打坐几分钟，现在没什么感觉了。

>最近几天睡眠还算充足，锻炼也比较规律，心态还可以，这真是太吓人了！

>刚才咳嗽的时候，也感到一下刺痛，今天晚上打坐并且早点休息

因为我很注重健康，尽量不加班，还常常锻炼，此前从来没出现过这种情况很奇怪，所以才会记录下来。

现在回想，这次可能的诱导因素是，我从三月底开始研究加密货币合约，因为没有经验，会因为亏钱患得患失，所以那一段时间可能心态不太好。
